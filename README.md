# 指标拆解树 — 归因分析能力建设方案

基于 `plugin-rsl-bi-indicator-tree` 代码库分析的归因分析能力建设方案展示页面。

## 在线预览

部署后通过 GitHub Pages 访问：`https://<username>.github.io/attribution-analysis-proposal/`

## 本地预览

直接用浏览器打开 `index.html` 即可，无需构建。

## 技术栈

- 纯 HTML / CSS / JavaScript，无框架依赖
- [Mermaid.js](https://mermaid.js.org/) — 架构图、时序图
- CSS 动画 — 滚动出现、时间线、瀑布图
- 深色主题，响应式布局
